# IBM-Cognos-Insurance-Analytics
Insurance analytics project using IBM Cognos including reports, dashboards, explorations and storytelling (Q1–Q6).




This project demonstrates insurance data analysis using IBM Cognos Analytics.

## Modules Covered

### Q1 – Reports
- Claims Settlement Report
- Policy Type Analysis
- Average Days to Settle

### Q2 – Regional Trends
- Fraud Indicator Report
- Regional Claim Insights

### Q3 – Product Analytics
- Monthly Claims vs Settlement Rate
- Product Performance Analysis

### Q4 – Deep Exploration
- Claim Analysis by Age Group
- Interactive Insights

### Q5 – Executive Dashboard
- Bar, Line, Pie Visualizations
- Claims Trends and Distribution

### Q6 – Data Storytelling
- Policy Portfolio Overview
- Claims Performance
- Fraud Analysis
- Agent & Branch Performance
- Customer Satisfaction Trends
- Action Plan
